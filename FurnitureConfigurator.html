3D Progress Bars Marketplace asset

Using Blender I created the 3D models with specific UV mapping which I then used inside Shadergraph to set the Alpha value of the different bars.

I then created specific scripts in Unity with the idea for the progress bars to be completely customizable from a main script. These scripts allow changing colors, animation curves, and other settings.

Project Idea

The idea came during the development of a 3D classroom where the professor had the need to show the classroom a video. I wanted to create a 3D video player with a progress bar, however, searching on the Marketplace I did not find anything that suited my needs. Thus, I had to create it myself and I turned the feature into a marketplace asset.
