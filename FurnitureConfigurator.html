DasGanzeLeben furniture configurator

This project was commissioned by the Realer company which, in turn, received it from DasGanzeLeben.
I developed the whole project starting from July 2022 and finished it in January 2023.

The application consists of different modules that the company DasGanzeLeben produces which can be placed in the scene based on real constraints.
The user can modify each module based on their needs by adding or removing them, moving them or changing material.

My role

I developed the whole application which consists of the configurator as well as some additional functionalities, such as PDF generation where the user can download a PDF based on their configuration, localization where the user can change choose between german, english or italian, Save and Load functionality based on a custom generated string.

I also developed the User Interface based on design directions with functionalities such as dynamic movement of elements simulating an accordion list.

DasGanzeLeben, the company who commissioned the project, also requested that some data could be modified via an excel file in order to always keep prices, dimensions and names up to date easily.

Some of the functionalities

One of the functionalities I had most fun implementing was the Accordion style UI where I replicated the movement of a dropdown menu. To achieve this effect I used unity's UI mask which values are changed via a coroutine in a C# script.

The client wanted the possibility to create a PDF based on the user's configuration. In order to achieve this functionality I used Unity's JsLib which passes the values from the WebGL application to a Javascript function to correctly print the PDF.
