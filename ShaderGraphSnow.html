Snow shader using Shadergraph and C#

Project information

  • Category: Personal project
  • Project date: June 2023
  • Live Demo: Demo

Project idea and how it works

The project was developed after learning the different functionalities of Unity's Shadergraph.

I created a highly subdivided plane in Blender and imported it inside Unity. After that, I applied a normal map and a noise height map to create the snow.
Then I applied another height map to the first one based on the player's previous position: by moving the player draws on the second height map using a custom mask texture in order to lower the height of the snow by applying the height to the vertices, thus creating a vertex displacement shader. Since the values are added in time, I managed to achieve the effect of the snow lowering more and more after each time the player walks on it.

Other effects

Sparkles

By creating two noise textures with different scales, applying a high power value (10-13) and then passing them throgh a step function I managed to get the sparkling effect. The sparkles however are not always visible. The effect was achieved by using the screen position as UV for one noise texture and then multiplying the result by the other noise texture, thus hiding the sparkles only sometimes and only when the player moves the camera.

I also wanted the sparkles to shine like they do in reality so I searched on the internet when do this sparkling effect is visible the most. After learning that the sparkles are mostrly visible when the light source is in front of the viewer I proceeded to create this effect by computing a dot product between the view direction and the main light direction; remapping it from (-1, 1) to (0,1) and multiplying the effect by this value
